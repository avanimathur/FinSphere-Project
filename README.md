
# 💰 FinSphere

A full-stack personal finance web application designed to help users manage their money, track expenses, and improve financial literacy.


### 🚀 Features

### 🔐 Authentication
- User Signup & Login (JWT-based authentication)
- Secure routes with protected access

### 👤 Profile Management
- Store user details (college, company, role, salary, city, etc.)
- Update & delete profile

### 📊 Budget Tracker
- Add income & expenses
- View transaction history
- Auto calculation of:
  - Total Income
  - Total Expenses
  - Savings

### 📝 Blog System
- Create blogs
- Read blogs (slug-based routing)
- Delete blogs
- Category filtering (Salary, Tax, Saving, Investment)
- Search functionality

### 📚 Financial Education
- Learn key financial concepts
- Beginner-friendly UI


## 🛠 Tech Stack

### Frontend
- HTML
- CSS
- JavaScript (Vanilla JS)

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Other Tools
- JWT (Authentication)
- REST APIs
- dotenv (Environment variables)
